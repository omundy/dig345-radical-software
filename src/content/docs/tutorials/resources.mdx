## Frontend

- [Vue.js](https://vuejs.org/) - Reactive frontend library (or [Nuxt.js](https://nuxtjs.org)) 
- [React.js](https://reactjs.org/) - Build user interfaces (or [Next.js](https://nextjs.org/)) 
- [jQuery](https://jquery.com/) & [jQuery UI](https://jqueryui.com/) - Frontend libraries

## Data

- [D3.js](https://d3js.org/) - Visualize data
- [Chart.js](https://www.chartjs.org/) - Create charts and represent data
- [Faker.js](https://fakerjs.dev/) - Generate fake (but realistic) data

## Mapping 

- [Leaflet.js](https://leafletjs.com/) - Create interactive maps
- [OpenLayers](https://openlayers.org/) - Interactive maps, geographic data 
- [Mapbox GL JS](https://www.mapbox.com/mapbox-gljs) - Interactive vector maps, etc. 


## Sound / Time

- [Tone.js](https://tonejs.github.io/) - Create interactive music in the browser.
- [Luxon 3.x](https://moment.github.io/luxon/#/) - Library for working with date & time
- https://github.com/aagentah/nw_wrld


## Visuals

- [Anime.js](https://animejs.com/) - Javascript animation engine
- [Three.js](https://threejs.org/) - Create animations, games, 3D, etc.
- [Paper.js](http://paperjs.org/) - A vector graphics scripting framework 
- [P5.js](https://p5js.org/) - Create animations, games, etc. 
- [A-Frame](https://aframe.io/) - Build VR experiences
- [Pts.js](https://ptsjs.org/) - Visualization and creative coding


## Applications 

- Desktop
    - [Electron](https://www.electronjs.org/) - Build cross-platform desktop apps with JavaScript, HTML, and CSS
    - [Tauri](https://v2.tauri.app/) - Create small, fast, secure, cross-platform applications (and mobile?) 
    - See also: [Electron vs. Tauri](https://www.dolthub.com/blog/2025-11-13-electron-vs-tauri/), 2025
- Mobile 
    - [Expo](https://expo.dev/) - A full-stack React Native framework
    - [React Native](https://reactnative.dev/) - Create native apps for Android, iOS, and more
