## React Native

While complex, [React Native](https://reactnative.dev/) is considered to be one of the most flexible options developers to build cross-platform, natively rendered applications. 
- Apps can run on iOS, Android, Windows, and macOS from a single codebase.
- A JS thread handles logic, state, and component structure, passing rendering updates using Fabric to the native UI thread which draws components on the screen.


### React Native Basics 

For more information on React see Codecademy's [Learn React Native](https://www.codecademy.com/enrolled/courses/learn-react-native) and React Native's [Learn the Basics](https://reactnative.dev/docs/tutorial) 




```js 
// App.tsx
import React from 'react';
import { View, Text } from 'react-native';

export default function App() {
  return (
    <View>
      <Text style={{}}>Welcome to React Native!</Text>   
    </View>
  );
}
```
<figure>
Views are foundational to mobile UI. In React Native, the `<View>` component provides structure and a wrapper for other UI elements. In this basic example, we import the View and Text components, then use them inside the return value from `App()`
</figure>




```js 
import { useState, useEffect } from 'react';
import { View, Text, Button } from 'react-native';

export default function Counter() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    // This runs after the first render, and when the count changes
    console.log(`Count changed to: ${count}`);
  }, [count]);

  return (
    <View>
      <Text>Count: {count}</Text>
      <Button title="Increment" onPress={() => setCount(count + 1)} />
    </View>
  );
}
```




## Expo 

[Expo](https://expo.dev/) is a framework, platform, and a set of tools for different stages of React Native app development:

- Expo Go: A client app for testing without building
- Expo SDK: Set of prebuilt native modules
- Expo CLI: Command-line tools for project management
- [Expo Snack](https://snack.expo.dev/): Online React Native playground for prototyping and sharing React Native code
- [EAS (Expo Application Services)](https://docs.expo.dev/build/introduction/): Cloud-based tools for app builds and updates
